Hover States

A shortcut to apply styling changes to the &:hover, &:focus, &:active states in one go.


/* Input */
a {
  color: blue;

  @include hover-states {
    color: red;
  }
}

/* Output */
a {
  color: blue;
}
a:hover,
a:focus,
a:active {
  color: red;
}
