Buttons

The .btn class can be applied to any element. For the examples below, an anchor tag is used.

A small plain button <a class="btn plain small">Text</a>

A yellow button <a class="btn yellow">Text</a>

A blue tiny button <a class="btn blue tiny">Text</a>
