Notes

Code has been indented using spaces with a tab-width of two.

Sass files are combined to styles.min.css using CodeKit.

CSS is written following the mobile-first approach, declaring styles for small viewports first and then making adjustments for larger viewports.

The scripts.js file is set up in CodeKit to import all used plugins at the top and is then concatenated and minified to scripts.min.js.
