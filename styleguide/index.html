Font icons

The font icons have been generated using IcoMoon.

Editing the font

To edit the font, follow these steps:

  1. Open the IcoMoon app in your browser.
  2. Click on the database icon at the bottom, and select Load Session.
  3. Select the sap-icons.json file from the assets/source-files folder. You'll see the used icons highlighted within the different libraries.
  4. To add another icon, either select an existing one from on of the libraries (currently IcoMoon and Font Awesome), or upload a custom icon by importing the .svg file.
  5. When you are done, click the Font button at the bottom and on the next page click the Download button.
  6. Click on the database icon at the bottom, and select Store Session and replace the old session file with this one in the assets/source-files folder.
  7. Use the new font files you've downloaded and replace the old ones.
  8. Update the icon classes within the _03elements.less file.
